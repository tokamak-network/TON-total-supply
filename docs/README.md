## 📊 Generated Data

The project generates 6 CSV files tracking different aspects of TON supply:
1. Block number mapping
2. Staked TON amounts
3. Burned TON amounts
4. DAO locked/spent TON
5. Burned seignorage
6. Reduced seignorage
